# dash-client
Client for DASH Video Streaming in Python

Code Modified from https://github.com/jancc/AStream

Commands:

    cd client
    
    ./dash_client.py -h
    
    MPD_URL=http://128.110.219.88/dash.mpd
    
    time python3 dash_client.py -m $MPD_URL -p basic -d
